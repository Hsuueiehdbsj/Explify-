‚ö° Quick Start - Local Development

Get Explify running on your machine in 3 simple steps:

Step 1: Extract & Setup

unzip explify-project.zip && cd explify && chmod +x setup.sh && ./setup.sh

Step 2: Configure API Key

  • Get your free API key: Google AI Studio
  • Run: cp .env.example .env
  • Edit .env and add your key

Step 3: Launch!

source explify_env/bin/activate && streamlit run explify_app.py

‚ú® Your app will open at http://localhost:8501
